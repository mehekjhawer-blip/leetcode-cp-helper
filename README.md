# LeetCode CP Helper
LeetCode CP Helper — Your Ultimate Competitive Programming Companion for VS Code
LeetCode CP Helper is a free, open-source Visual Studio Code extension built by a developer, for developers. Designed to eliminate the constant friction of switching between your browser and editor while solving problems, this extension brings the entire LeetCode experience directly into your coding environment — so you can focus on what truly matters: solving problems and improving your skills.
Whether you are a student preparing for campus placements, a professional brushing up for FAANG interviews, or a competitive programmer grinding daily problems, this extension is built to make your workflow faster, smarter, and more organized.

✨ Features & Commands

 Every feature below was born from a real frustration faced while solving problems competitively.

Commands for efficient usage :

🔍 "CP Helper: Fetch LeetCode Problem"
Simply type a problem slug (e.g. two-sum) and the extension instantly fetches the full problem, parses it, and opens a ready-to-code file with the problem description as a comment and the function signature pre-filled. No more copy-pasting from the browser. No more formatting boilerplate manually. Just open VS Code and start solving.

📚 "CP Helper: Browse Problems"
Not sure what to solve next? The built-in problem browser lets you filter problems by market importance category — FAANG Essentials, High Frequency Interview Questions, Competitive Programming Classics, and Startup Interview favorites — as well as by difficulty level (Easy, Medium, Hard). Every problem is curated and categorized so you always know what to practice based on your goal.

⚡ "CP Helper: Generate Boilerplate"
Supports C++, Python, Java, and JavaScript. The extension automatically generates clean, well-structured boilerplate code tailored to the problem's function signature, saving you the repetitive setup work every single time.

⏱ "CP Helper: Start Timer / Stop Timer"
Simulate real interview and contest conditions with the built-in countdown timer. Once started, a live countdown appears directly in your VS Code status bar so you always know how much time you have left. The timer turns red in the final 60 seconds to keep you alert. It is a small feature that makes a big difference in building the time-awareness habits that competitive programming demands.

📊 'CP Helper: Show Dashboard"
Your personal problem-solving dashboard, built right inside VS Code. View all the problems you have fetched, their difficulty levels, whether you have solved them, the date you solved them, and any notes you have added. It is a clean, dark-themed webview panel that gives you a bird's eye view of your progress without leaving your editor.

✅ "CP Helper: Mark Problem Solved"
Once you crack a problem, mark it as solved with a single command. The dashboard updates instantly, giving you that satisfying sense of progress and helping you track which problems you have already covered.

📝 "CP Helper: Add Note to Problem"
After solving a problem, add a quick note — the approach you used, the time complexity, edge cases to remember, or anything else worth noting. These notes are saved persistently and visible in your dashboard, turning your problem history into a personal revision guide.

🚀 "CP Helper: Submit Solution"
Submit your solution directly to LeetCode from inside VS Code without opening the browser. Supports all major languages and provides instant feedback on whether your submission went through successfully.

🧪 "CP Helper: Run Test Cases"
Compile and run your code locally with a single command. The extension opens a terminal, compiles your file (g++ for C++, python3 for Python, javac for Java), and runs it immediately — letting you test your logic quickly before submitting.

💡 Why LeetCode CP Helper?
Most developers who practice on LeetCode follow the same inefficient loop — open browser, read problem, copy function signature, switch to editor, paste, format, code, switch back to browser, submit. This extension breaks that loop entirely.
Every minute saved on setup is a minute spent on actual thinking. Every problem tracked in the dashboard is one less thing to remember. Every note saved is one more insight retained. Cumulatively, these small improvements compound into a significantly better and more focused practice experience.
This extension was built out of a genuine need felt during competitive programming practice, and it is shared freely with the community in the hope that it helps others on the same journey.


## 🔧 Installation

This extension is published on **Open VSX Registry** and can be installed in:
- **VSCodium** — search `leetcode-cp-helper` in Extensions panel
- **VS Code** — install manually via the `.vsix` file or run:
```bash
code --install-extension MehekJhawer.leetcode-cp-helper
```

Or download directly from:
🔗 https://open-vsx.org/extension/MehekJhawer/leetcode-cp-helper
